Artificial Intelligence Chatbot

Project Description

This project is a Java-based AI Chatbot developed using Java Swing for the graphical user interface. The chatbot allows users to interact through a simple chat window and provides responses based on predefined rules and frequently asked questions.

Features

- Interactive GUI using Java Swing
- Real-time user-chatbot communication
- Natural Language Processing (NLP) using text preprocessing
- Rule-based response generation
- FAQ-based knowledge base
- Easy to use

Technologies Used

- Java
- Java Swing
- HashMap
- Object-Oriented Programming (OOP)

NLP Techniques Used

- Lowercase conversion
- Removal of special characters
- Keyword matching

AI Technique Used

Rule-Based Artificial Intelligence with FAQ response mapping.

Project Structure

- ChatBotGUI.java
- ChatBotLogic.java
- FAQData.java

How to Run

1. Open the project in VS Code.
2. Compile the Java files:
   javac *.java
3. Run the application:
   java ChatBotGUI
